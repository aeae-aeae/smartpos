# 🏪 SmartPOS Pro

ระบบขายหน้าร้าน (POS) + จัดการลูกค้า + การจัดส่ง **ครบวงจร**

**🚀 ใช้งานได้ทันที ฟรี ไม่ต้อง Server!**

---

## ✨ ฟีเจอร์หลัก

- 📱 **ขายหน้าร้าน (POS)** - เลือกสินค้า ตะกร้า ชำระเงิน พิมพ์ใบเสร็จ
- 👥 **จัดการลูกค้า** - บันทึกข้อมูล ชื่อ เบอร์ ที่อยู่
- 🚚 **จัดการการจัดส่ง** - Tracking สถานะจัดส่ง
- 📊 **รายงานสรุป** - ยอดขาย กำไร รายได้รวม
- ⚙️ **ตั้งค่า** - VAT 7% (เปิด/ปิด) Export/Import ข้อมูล

## 🚀 วิธีใช้งาน

### **🌐 Deploy ไป GitHub Pages (แนะนำ!)**

อ่านคู่มือละเอียดใน: **[DEPLOY-GUIDE.md](DEPLOY-GUIDE.md)**

**ขั้นตอนสรุป:**
1. สมัคร GitHub (ฟรี)
2. สร้าง Repository ใหม่
3. อัพโหลดไฟล์ `index.html`
4. เปิด Settings → Pages → เลือก Branch
5. เสร็จ! ได้ URL: `https://yourusername.github.io/smartpos/`

### **💻 ใช้งานแบบ Offline**
1. ดาวน์โหลด `index.html`
2. Double-click เปิดด้วย Chrome/Safari
3. ใช้งานได้เลย!

## 💾 Backup & Sync ข้อมูล

- **Export:** ตั้งค่า → ส่งออกข้อมูล → ได้ไฟล์ .json
- **Import:** ตั้งค่า → นำเข้าข้อมูล → เลือกไฟล์
- **แชร์:** ส่งไฟล์ .json ผ่าน LINE/Email
